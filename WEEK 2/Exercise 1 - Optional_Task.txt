# 💡 Ejercicios Extra

## 1. Ordenar dos números

**Descripción:** Se pide 2 números al usuario (`primero` y `segundo`) y se ordenan de menor a mayor.

**Ejemplos:**  
- A: 56, B: 32 → A: 32, B: 56  
- A: 24, B: 76 → A: 24, B: 76  
- A: 45, B: 12 → A: 12, B: 45

**Respuesta en pseudocódigo:**

```
Inicio

1. Definir primero
2. Definir segundo
3. Definir A
4. Definir B
5. Mostrar “Ingrese el primer numero: ”
6. Pedir primero
7. Mostrar “Ingrese el segundo numero: ”
8. Pedir segundo
9. Si (segundo < primero):
    1. A = segundo
    2. B = primero
   Sino:
    1. A = primero
    2. B = segundo
FinSi
10. Mostrar A
11. Mostrar B

Fin
```
